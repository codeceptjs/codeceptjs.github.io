# ExpectHelper

This helper allows performing assertions based on Chai.

# Examples

Zero-configuration when paired with other helpers like REST, Playwright:

// inside codecept.conf.js
{
  helpers: {
    Playwright: {...},
    Expect: {},
  }
}
